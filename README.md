# 🧸 Veloura Squishy Store — Website Toko Online

Website toko online resmi **Veloura Squishy Store**, platform penjualan squishy berkualitas tinggi yang menyediakan berbagai produk lucu dan menarik dengan harga kompetitif. Website ini mencakup katalog produk, program kemitraan bisnis, promo kartu kredit, dan pusat layanan pelanggan.

🔗 **Live Demo:** [squishy-store.vercel.app](https://squishy-store.vercel.app)

---

## ✨ Fitur Utama

### 🏠 Beranda (Home)
- Hero slider dengan banner produk unggulan
- Informasi singkat dan tagline toko

### 🛍️ Katalog Produk
- Filter produk berdasarkan kategori: **Squishy Food**, **Squishy Animal**, dan **Best Seller**
- Badge label produk: **Hot Deal**, **New**, **Limited**
- Modal detail produk dengan spesifikasi lengkap
- Tombol **Buy Now** langsung dari halaman produk

### 🤝 Program Mitra Bisnis
- Informasi keuntungan bergabung sebagai mitra
- Daftar mitra terpercaya beserta testimoni
- Form / link untuk bergabung sebagai mitra baru

### 💳 Promo Kartu Kredit
- Promo cashback dari berbagai bank (DBS, BRI, OCBC NISP, Bank Mayapada)
- Program cicilan 0% dari BCA, Mandiri, BNI, dan Citibank
- Form pengajuan kartu kredit langsung dari website

### 🎧 Service Center / Pusat Layanan
- Lacak status pengiriman pesanan
- Kebijakan pengembalian produk (7 hari)
- Multi-channel dukungan: Telepon, WhatsApp, dan Email
- Halaman FAQ lengkap (cara pesan, pengiriman, pembayaran, retur)
- Form pengiriman pesan ke tim CS

### 📞 Contact Us
- Informasi kontak lengkap toko
- Lokasi: Bandung, Indonesia
